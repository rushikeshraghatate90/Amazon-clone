# Amazon Clone

Welcome to the **Amazon Clone Project**! This project is a simple recreation of the **Amazon.com** look and feel using **HTML** and **CSS**. It’s designed to mimic the UI of Amazon’s website, showcasing various web design and layout techniques. 🌐✨

---

## 🌟 Features

### 1. **Responsive Header**
- **Navigation Bar:** Includes the logo, address section, search bar, sign-in options, and cart. 🛒
- **Hover Effects:** Interactive border highlights when hovering over elements for better user experience. ✨

### 2. **Pannel Section**
- Provides quick navigation options like "Today's Deals," "Customer Services," and more. 📋
- Includes a collapsible menu with an icon. 📂

### 3. **Hero Section**
- A large banner with a background image representing the main feature of the page. 🖼️
- Includes a notification message about shopping on Amazon India with a clickable link. 🌍

### 4. **Shop Section**
- Displays categories like **Clothes**, **Electronics**, **Furniture**, and more.
- Each category is represented by a styled box with a title, image, and link to explore further. 🛍️
- Flexible layout using **Flexbox** for a responsive design. 📱💻

### 5. **Footer**
- Organized into panels:
  - "Back to Top" button. ⏫
  - Multiple sections with links to resources like Careers, Blog, Investor Relations, etc. 🔗
  - Footer branding with a logo section. 🔵

## 🚀 How to Run
1. Clone the repository or download the files.
2. Ensure you have the required **images** (e.g., `amazon_logo.png.jpg`, `hero_image.jpg`, and category images like `box1_image.jpg`).
3. Open the `index.html` file in your browser.

## 🎨 Design Highlights
- **Hover Effects:** Adds interactivity and enhances the user experience.
- **Flexbox Layout:** Used extensively for responsive and aligned sections.
